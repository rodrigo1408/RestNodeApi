Backend server for any uses

🚀 Technologies
This project was developed with the following technologies:

Node.js

Install Back-end 

# Install dependencies
$ npm install
$ npm sequelize-cli 

# Run Migrates
$ npm run knex:migrate

# Run migrations
$ npx sequelize-cli db:migrate

# Start server
$ node start

# running on port 3000
